# 俄罗斯方块 (Tetris)

纯 HTML/CSS/JavaScript 实现的经典俄罗斯方块游戏。

## 操作方式

| 按键 | 功能 |
|------|------|
| ← → 或 A D | 左右移动 |
| ↑ 或 W | 旋转 |
| ↓ 或 S | 加速下落 |
| 空格 | 硬降（直接落底） |
| P | 暂停/继续 |

## 特性

- 7种经典方块（I、J、L、O、S、T、Z）
- 分数、行数、等级系统
- 下一个方块预览
- 幽灵方块（落点预览）
- 暂停功能
- 响应式设计，支持移动端

## 使用

直接在浏览器中打开 `tetris.html` 即可开始游戏。

[在线体验](https://nothing0961.github.io/little-game/tetris.html)
